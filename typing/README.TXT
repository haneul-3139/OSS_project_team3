## typing_main @@@@@@@@@@@ 간단한 설명 @@@@@@@@@@@@@@@@@@@@
로그인창 구현 , 입력받은 정보를 DB와 비교 가능. 
회원가입창 구현, 입력받은 정보를 DB에 저장함.
로그인 완료시에 새로운 게임GUI오픈하게하려함.

##game_execute @@@@@@@@@@@ 설명 @@@@@@@@@@@@@@@@@@@@@@@@@@@
tkinter모듈을 이용해서 프레임(게임 실행 창) 생성 함.
랜덤으로 생성되는 단어, 소요시간, 정답개수, 입력 창을 프레임에 올림.
한 사용자 당 10번의 랜덤 단어가 보여지고, 10번의 실행이 끝나면 게임 창이 닫히고, 결과창이 보여짐.
결과창은 맞힌개수와 소요시간이 보여짐.
